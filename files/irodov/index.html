Файлы / Решённые задачи из задачника Иродова

В течение трёх лет обучения — с первого по третий курс — активно используется практических на занятиях по общей физики задачник Иродова. Но не всегда получается решить задачу, поэтому именно для этого и создан этот раздел: помочь в решении задач. Но будьте аккуратнее — всё-таки сначала подумайте над задачей сами, а потом смотрите решение.

Самые важные материалы

Все решённые задачи из Иродова (на английском языке1):

  • Решённые задачи из Иродова: механика, термодинамика, молекулярная физика, электродинамика (электричество) (~7.5 Mb)
  • Решённые задачи из Иродова: колебания и волны, оптика, атомная и ядерная физика (~6.5 Mb)

Необходимые учебники и задачники:

  • Задачник Иродова за 1981 год на английском языке (нумерация печатных решённых задач именно по этому учебнику, ~3.9 Mb)
  • Задачник Иродова за 1979 год на русском языке (нумерация печатных решённых задач именно по этому учебнику, ~3.6 Mb)
  • Учебник И.Е.Иродова: механика (~1.9 Mb)
  • Учебник И.Е.Иродова: основные законы электромагнетизма (~2.1 Mb)
  • Учебник И.Е.Иродова: волновые процессы, основные законы (~1.5 Mb)
  • Учебник И.Е.Иродова: квантовая физика, основные законы (~1.7 Mb)
  • Учебник И.Е.Иродова: физика макросистем, основные законы (~1.3 Mb)
