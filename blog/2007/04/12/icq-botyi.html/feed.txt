11) !rss [rss-новость] - Заявка на добавление Rss-Новости

21) !valute - Курсы валют других стран
22) !pm [uin] - Примари
23) !stat - Статистика Ботов
24) !sat [спутник] - Транспондеры спутников
25) !bash - Случайная цитата с bash.org.ru
Помощь по функциям:
1) !help - Помощь по функциям Бота
2) !news - Помощь по источникам и получению новостей
3) !rub - Помощь по темам новостей
4) !str - Помощь по работе с текстом
5) !alias - Список алиасов
6) !tvchan - Список доступных каналов
7) !bot - Список всех Ботов
8) !satlist - Список доступных спутников
9) !about - Информация о проекте FireBOT
Команды [F-BOT] Translate
Бот поддерживает автоопределение языка перевода (Eng&lt;-&gt;Rus - Rus&lt;-&gt;Eng). Для этого просто введите нужный текст, язык будет распознан по первому слову в тексте
![с какого][на какой] [текст]
--------------
u - Украинский
r - Русский
e - Английский
g - Немецкий
f - Французский
l - Латышский
--------------
Пример:
--------------
!re Привет
или просто же
Привет
